# ECE NTUA Student Portal

This repository hosts an experimental student portal for the School of Electrical and Computer Engineering at the National Technical University of Athens. It uses a Next.js 15 app located in the `portal/` directory and currently provides:

- University announcements
- Lab partner matching
- A downloadable semester calendar
- Past exam papers
- A social network area for signing in with Google or IEEE credentials (coming soon)

## Development

```bash
cd portal
npm install
npm run dev
```

## Inspiration
